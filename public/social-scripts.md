## Bonus: Platform-Specific Formatting Tips

**TikTok**: Start with movement/action. Hook in first 0.5 seconds. Use trending sounds when appropriate.

**Instagram Reels**: Use text overlays. Strong visual hook. End with clear CTA.

**YouTube Shorts**: Front-load the hook. Promise value immediately. Use pattern interrupts.

**Pinterest**: SEO-rich descriptions. Vertical images. Save-worthy content.

**Twitter/X**: Thread format for depth. Single punchy tweets for reach. Engage in replies.

**Facebook**: Longer form acceptable. Community-focused language. Personal stories perform well.
